# 英语学习平台

一个为初学者设计的交互式英语学习平台。

## 功能特性

- 📚 **单词学习** - 学习常用英语单词和短语
- 🎯 **测验练习** - 通过选择题测试你的学习成果
- 📊 **进度追踪** - 记录学习进度和成绩
- 👤 **用户管理** - 注册、登录、个人学习数据保存
- 🎨 **友好界面** - 简洁易用的学习界面

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript
- **后端**: Node.js + Express
- **数据库**: SQLite
- **认证**: JWT

## 快速开始

### 1. 克隆项目
```bash
git clone https://github.com/qingchaohuang06-bot/english-learning-platform.git
cd english-learning-platform
```

### 2. 安装依赖
```bash
npm install
```

### 3. 启动服务器
```bash
npm start
```

### 4. 打开浏览器
访问 `http://localhost:3000`

## 学习内容

### 第一阶段：基础词汇

### 第二阶段：中级词汇
